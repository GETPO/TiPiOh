# TiPiOh!
T.P.O (Time, Place, Occasion) 기반 SNS 앱

## Installation & build
1. frontend 폴더로 이동
```bash
cd TiPiOh/frontend
```
2. npm 모듈 설치
```bash
npm install
```
3. expo 실행
```bash
expo start -c
```
